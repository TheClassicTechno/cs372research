KEY OBSERVATIONS
================================================================================

1. INTERVENTION NEVER ACTUALLY FIRED
   Despite the intervention config having threshold=0.6, the intervention_fired
   flag is false for ALL runs. This means the role-specific nudge prompts were
   NEVER injected during any revision phase. The difference between configs
   must therefore come from OTHER config differences (the threshold=0.0 baseline
   uses "placeholder" nudge text vs threshold=0.6 intervention with full prompts,
   but since neither actually fired, the observed differences are stochastic).

2. FINANCIAL PERFORMANCE: MARGINAL INTERVENTION EDGE
   Intervention wins 4/6 on total return with +0.22 pp average advantage.
   This is a SMALL effect with high scenario variance. Not statistically
   significant with n=6.

3. CRIT SCORES: INTERVENTION CONSISTENTLY HIGHER
   Intervention wins 4/6 on rho_bar with +0.014 average improvement. The
   2023Q2_rand8 scenario shows the biggest gain (+0.073). This is interesting
   but since intervention never fired, this is likely noise.

4. DIVERGENCE: INTERVENTION MAINTAINS MORE
   Intervention maintains higher JS divergence in 4/6 cases, with +0.023
   average gain. Higher JS = agents maintained more independent views.

5. NO INTERVENTION EFFECT IS REAL -- ALL DIFFERENCES ARE STOCHASTIC
   Since the intervention mechanism never activated (intervention_fired=false
   for all 16 runs), any differences between baseline and intervention configs
   are due to LLM stochasticity, not the treatment. The configs are functionally
   identical in terms of what actually happened during the runs.

6. BOTH CONFIGS USE 3 AGENTS (macro, technical, value)
   This is different from ablation 7 (2 agents) and ablation 8 (3 agents).
   Ablation 11 adds the value agent perspective.

7. LARGE UNIVERSE SCENARIOS SHOW HIGH DIVERGENCE
   scenario_2023Q2 (23 tickers) has JS > 0.4, while narrow tech-focused
   scenarios (5-6 tickers) cluster around 0.08-0.15. Universe size strongly
   drives divergence.

   pairs will be available once all complete.
